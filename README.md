# ☀️ Solar Energy Application (Солнечная энергия): Калькулятор для частных домов

Приложение для подбора оборудования и расчета экономической эффективности солнечных электростанций. Помогает владельцам домов оценить окупаемость перехода на возобновляемую энергию.

## 🛠 Основные услуги и функции

Проект автоматизирует процесс проектирования системы электроснабжения и заказ необходимого оборудования.

Услуги - устройства для частных домов (солнечные батареи, аккумуляторы), заявки - расчет генерации и сбережения для указанного набора устройств и широты.

### 1. Каталог устройств
*   **Солнечные батареи:** Выбор панелей по типу (монокристаллические, поликристаллические) и мощности.
*   **Аккумуляторы:** Подбор емкости для автономного или резервного питания.

### 2. Умные заявки и расчеты
Главная функция — расчет генерации на основе входных данных:
*   **Широта (Геопозиция):** Учет угла падения солнечных лучей и инсоляции в конкретном регионе.
*   **Конфигурация:** Расчет для выбранного набора панелей и емкости АКБ.
*   **Экономия:** Прогноз сбережения средств на электроэнергии и расчет срока окупаемости (ROI).

## 📊 Как работает алгоритм
Есть ключевой процесс, в котором пользователь оформляет заявки, в которой может быть несколько услуг.

Также есть модератор, который может редактировать список услуг и одобрять заявки.

Приложение использует данные о солнечной активности для указанной широты и сопоставляет их с характеристиками выбранных устройств:
1. Пользователь выбирает **набор оборудования**.
2. Указывает **географическую широту** (или выбирает город).
3. Система выдает **прогноз выработки в кВт⋅ч** по месяцам и потенциальную экономию в денежном выражении.
